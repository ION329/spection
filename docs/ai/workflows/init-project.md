# Workflow: init-project

## Objective

Initialize a new software project using the AI‑Orchestrated Engineering
methodology.

------------------------------------------------------------------------

## Phase 1 --- Product Discovery

Agent: product-strategist

Generate:

docs/spec/product/

Documents: - product.md - problem.md - personas.md - use-cases.md -
value-proposition.md

------------------------------------------------------------------------

## Phase 2 --- System Architecture

Agent: software-architect

Generate:

docs/spec/architecture/

Documents: - system-overview.md - tech-stack.md - data-model.md -
security.md - scalability.md

------------------------------------------------------------------------

## Phase 3 --- Feature Definition

Agent: systems-engineer

Generate:

docs/spec/features/

Each feature receives its own specification.

------------------------------------------------------------------------

## Phase 4 --- Engineering Specifications

Agent: lead-engineer

Generate:

docs/spec/engineering/

Documents: - api-contracts.md - database-schema.md - ui-components.md -
state-management.md

------------------------------------------------------------------------

## Phase 5 --- Roadmap

Agent: product-strategist

Generate:

docs/spec/roadmap/

Define milestones and implementation order.

------------------------------------------------------------------------

## Result

After executing the workflow the project contains:

docs/ ai/ spec/ public/ lab/

With product definition, architecture, and technical specifications
ready for implementation.
